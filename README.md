# ختمة | Katmah

منصة ويب تساعد المستخدم على تنظيم **الحفظ** و**المراجعة** و**التلاوة** من خلال خطط يومية واضحة، مع متابعة التقدم وسجل إنجاز بسيط ومرتب.

## نبذة عن المشروع

**ختمة** هي منصة ويب صُممت لمساعدة المستخدم على الاستمرار في رحلته مع القرآن، من خلال تحويل الأهداف الكبيرة إلى خطط يومية قابلة للتنفيذ.

تتيح المنصة للمستخدم إنشاء خطة ختمة منظمة، ومتابعة التقدم اليومي، ومراجعة سجل الإنجازات، والاطلاع على التاريخ الكامل بشكل واضح وسهل.

* متابعة التقدم اليومي
* سجل للإنجازات

* لوحة تحكم للمستخدم

## التثبيت

```bash
git clone https://github.com/BanderA400/Katmah---.git
cd Katmah---
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
composer run dev
```

## أوامر مفيدة

### تشغيل الاختبارات


### بناء ملفات الواجهة للإنتاج


## هدف المشروع
